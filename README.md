# Emerald PT-BR HGSS

Projeto de overhaul visual do Pokémon Emerald em português brasileiro.

A pipeline clona a base pública `lucmsilva651/esmeralda-ptbr`, aplica o delta visual da v1.0 e gera um artefato de build para testes pessoais.

## Estado

- 386 sprites de batalha integrados
- água, margens, cachoeira e surf reformulados
- Littleroot, Oldale, Route 101 e Petalburg Woods com tilesets isolados
- build automatizada em GitHub Actions
